# Selíni — Creative Studio

The site for **Selíni**, a creative studio for films, ads and music videos, led by Sahar Machluf Ziv.

Two faces, one site:

- `/` — the studio: philosophy, services, selected work, contact. For clients who want a full production.
- `/director` — Sahar as director: bio, skills, selected work. For clients who want Sahar on their project.

Design follows the Selíni Brand Book v1 (2026): terracotta / cream / olive ink / warm black palette, Cormorant Garamond display type, Quicksand body type, and the waxing-crescent mark.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

> Note: if the project path contains `&` (Windows), run Next directly:
> `node node_modules/next/dist/bin/next dev`
